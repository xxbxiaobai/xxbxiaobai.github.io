数组新增方法

new Set()

1.什么是Set()
  • Set是es6新增的数据结构，似于数组，但它的一大特性就是所有元素都是唯一的，没有重复的值，我们一般称为集合。
  • Set本身是一个构造函数，用来生成 Set 数据结构
2、增删改查方法

添加元素add

  • 添加某个值，返回 Set 结构本身，当添加实例中已经存在的元素，set不会进行处理添加

删除元素delete

  • 删除某个值，返回一个布尔值，表示删除是否成功

判断某元素是否存在has

  • 返回一个布尔值，判断该值是否为Set的成员

清除所有元素clear

  • 清除所有成员，没有返回值
3、遍历方法

遍历 keys()

  • 返回键名的遍历器，相等于返回键值遍历器values()

遍历 values()

  • 返回键值的遍历器

遍历 entries()

  • 返回键值对的遍历器

遍历 forEach()

  • 使用回调函数遍历每个成员
